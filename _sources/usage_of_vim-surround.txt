Usage of vim-surround
=====================

Installation
++++++++++++
.. code-block :: vim

   NeoBundle 'tpope/vim-surround'


Example
+++++++

editing html
------------
.. code-block :: html

   Hello, world!

select the region of `Hello, world!` and enter

  `S<h1>`

this gets below

.. code-block :: html

   <h1>Hello, world!</h1>
